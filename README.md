# 📊 Análise de Dados — Pedágios Federais (.Gov)

Este repositório contém um projeto completo de análise de dados sobre as praças de pedágio das rodovias federais concedidas no Brasil.  
O objetivo é transformar dados públicos em insights claros, visuais e úteis para entendimento de receita, tráfego e comportamento operacional.

# 🎯 Objetivo do Projeto

Criar um dashboard analítico no Power BI utilizando dados abertos do Governo Federal, permitindo visualizar:

- Receita por praça e concessionária  
- Volume de tráfego por período  
- Emissões de vale-pedágio  
- Comparações entre rodovias e praças  
- Indicadores operacionais relevantes  
- Análises temporais e comportamentais  

O foco é desenvolver um painel intuitivo, limpo e orientado a decisões.

Os dados foram extraídos do portal oficial de dados abertos do Governo Federal.

    Nome do Arquivo: receita_de_pedagio
3. **Emissões mensais do vale pedágio obrigatório** 
    Nome do Arquivo: vpo-vale-pedagio-obrigatorio
4. **Volume de tráfego equivalente por praça**
